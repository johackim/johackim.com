Si vous voulez naviguer sur le web sans avoir peur d'être traqué, voici les outils à utiliser.

## Firefox

[Firefox](https://mozilla.org/fr/firefox/new/) est un logiciel libre. Contrairement à Chrome, il ne contient pas de code propriétaire.

C'est le navigateur à privilégier si vous désirez naviguer sur le web librement.

Il existe une version pour Linux, Windows, macOS, Android, iOS, etc...

PS : Il existe aussi la version communautaire de Firefox -> [LibreWolf](https://librewolf.net/)

## uBlock Origin

Pour [[Bloquer les publicités|éviter les publicités intempestives]], le meilleur add-on pour ça, c'est [uBlock Origin](https://ublockorigin.com/fr).

Contrairement à Adblock, uBlock Origin ne se fait pas rémunérer par les géants du Web pour laisser des publicités affichées et il consomme moins de ressources (CPU et RAM).

## Ungoogled-chromium

Si comme moi vous trouvez que Firefox est plus lent que Chrome/Chromium, vous pouvez installer la version degooglisé de Chrome.

Vous pouvez l'[installer depuis cette page](https://ungoogled-software.github.io/ungoogled-chromium-binaries/).

Si vous êtes sur Linux, vous pouvez aussi utiliser [[Flatpak|Flatpak]] :

```bash
flat install com.github.Eloston.UngoogledChromium
```

PS : Le chrome store n'est pas disponible par défaut, il vous faudra [installer cette extension](https://github.com/NeverDecaf/chromium-web-store) manuellement pour l'utiliser.

## Libredirect

Si vous aimez le contenu de YouTube, Instagram, Twitter et compagnie mais que vous ne voulez pas être traqué lors de son visionnage vous pouvez utiliser l'[extension Libredirect](https://libredirect.github.io/).

Une fois installé, tous les contenus seront automatiquement démarrés sur des alternatives respectueuses de la vie privée :

- YouTube => [Invidious](https://github.com/iv-org/invidious)
- Twitter => [Nitter](https://github.com/zedeus/nitter)
- Instagram => [Bibliogram](https://sr.ht/~cadence/bibliogram/)
- Reddit => [Libreddit](https://github.com/spikecodes/libreddit)
- Imgur => [Rimgo](https://codeberg.org/video-prize-ranch/rimgo)
- Google => [SearX](https://searx.github.io/searx/)
- Translate => [SimplyTranslate](https://git.sr.ht/~metalune/simplytranslate_web)
- Maps => [OpenStreetMap](https://openstreetmap.org/)
