[Retrobat](https://retrobat.org) vous permet de jouer aux jeux vidéos de vos consoles préférés directement depuis votre ordinateur Windows.

## Installation de Retrobat

Pour l'installer, rendez-vous sur [la page de téléchargement](https://retrobat.org/download/) pour télécharger le fichier .exe d'installation.

Exécutez le fichier sur Windows et suivez les instructions d'installation.

## Utilisation de Retrobat

**Je vous conseille fortement de brancher et d'utiliser une manette avant de démarrer Retrobat**, mais au cas où, voici quelques raccourcis clavier à connaitre :

- Pour valider un choix, appuyez sur la touche `x`.
- Pour annuler une action, appuyez sur la touche `z`.
- Pour afficher le menu principal, appuyez sur la touche `Entrée`.
- Pour afficher le menu secondaire, appuyez sur la touche `<-- Retour arrière`.

[Rendez-vous ici pour consulter les autres raccourcis](https://github.com/kaylh/RetroBat/wiki/Default-Inputs).

Démarrez Retrobat, puis sélectionnez un des jeux disponibles sur la console de votre choix.

## Télécharger un jeu

Pour télécharger vos jeux vidéos, rendez-vous sur votre moteur de recherche préféré et tapez la recherche suivante :

```txt
<jeux video> <console> ROM
```

Par exemple, pour télécharger le jeu Mario Smash Football sur GameCube :

```txt
Mario Smash Football GameCube ROM
```

Ensuite, téléchargez votre ROM via un lien qui fonctionne parmi la liste des résultats.

PS : Sinon vous pouvez aussi vous rendre sur [Crocdb.net](https://crocdb.net) !

## Installer un jeu

Pour installer un jeu, il vous suffit de copier-coller vos ROMS dans le dossier `C:\RetroBat\roms`.

## Sauvegarder un jeu

Lorsque vous jouez, les sauvegardes sont enregistrées automatiquement dans le dossier `C:\RetroBat\saves`.

Pour les sauvegardes GameCube, rendez-vous dans l'émulateur Dolphin et faites un clique droit sur votre jeu puis aller dans le dossier de sauvegardes où se trouvent vos fichiers `.cgi`.

## Configurer la touche spéciale (hotkey) de votre manette

Pour configurer le hotkey de votre manette (ex : `select`), rendez-vous dans les paramètres des manettes et configurer une manette.

Cette touche spéciale vous permettra de gérer Retrobat et les émulateurs avec des raccourcis pour pouvoir quitter un jeu rapidement ou créer/charger une sauvegarde rapide.
