Une note atomique est une note qui ne peut pas être divisée sans perdre son sens.

Voici les caractéristiques d'une note atomique :

- Une note atomique est égale à une seule idée
- Une note atomique ne dépasse pas une page
- Une note atomique s'explique d'elle-même

Décomposez chaque note afin qu'elles deviennent indivisibles.

Une note atomique ne dépasse pas une page.

Elle doit être suffisamment précise pour être comprise sans contexte.
