[Etcher](https://etcher.balena.io/) est un logiciel de gravure d'image disque open-source.

Il vous permet de graver des images disque (.iso, .img, etc.) sur des supports de stockage (clés USB, cartes SD, etc.).

Utile pour créer une [[USB|clé USB]] bootable et installer un système d'exploitation (ex: Ubuntu, Windows, etc...) sur un ordinateur.

Une fois installé, il vous suffit de :

1. Télécharger votre image disque (exemple : [Ubuntu](https://www.ubuntu-fr.org/download/))
2. Sélectionner l'image disque à graver (exemple : `ubuntu-24.04-desktop-amd64.iso`)
3. Choisir votre support de stockage (clé USB, carte SD, etc.)
4. Lancer la gravure

Une fois la gravure terminée, vous pourrez brancher votre clé USB sur votre ordinateur pour installer votre nouveau système d'exploitation.

Je l'utilise souvent pour installer des distributions Linux comme [[Arch Linux]], [[DietPI]] ou autre.

Ce logiciel est disponible pour Windows, macOS et Linux.
