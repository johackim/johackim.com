[Whatweb](https://github.com/urbanadventurer/WhatWeb) est un logiciel en ligne de commande comme [wappalyzer](https://wappalyzer.com/) pour analyser les outils utilisés par un site internet.

C'est un outil qui peut être utilisé en [[Cybersécurité]] pendant la phase de [[Reconnaissance]] d'un pentest.

Pour l'installer sur Arch LInux, exécutez la commande suivante :

```bash
yay -S --noconfirm whatweb
```

Pour l'utiliser :

```bash
whatweb -v <website>
```
