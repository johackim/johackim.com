Pour connaitre le format d'une date, il existe l'outil en ligne [NSDateFormatter](https://nsdateformatter.com/).

Renseignez votre date (ex : `2021-01-01T10:00`) dans le champs `DATE INPUT` et le site retournera le format de la date -> `ISO 8601 format`.
