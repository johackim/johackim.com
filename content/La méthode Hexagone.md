J'ai récemment découvert via une [conférence TED](https://youtube.com/watch?v=fYnJPtEJj4s), une méthode très pratique qui permet d'explorer et de trouver de nouvelles idées à partir d'une idée déjà existante.

![La méthode Hexagone](https://i.imgur.com/RVNXz0p.png)

En partant d'une idée X il est possible d'en trouver une nouvelle via 6 manières différentes :

## 1. Xᵈ

Première manière de faire, on part avec une idée X d'un domaine Y et on essai de l'adapter dans un autre domaine.

**Formule** : X pour Y => X pour … ?

**Exemples** :

- Uber pour les taxis -> Uber pour les services de nettoyage à domicile
- Plateforme pour partager des photos (Flickr) -> Platforme pour partager des vidéos (YouTube)
- Des airbags pour les voitures -> Airbags pour les vélos

## 2. X + Y

Deuxième manière de faire, on prend une idée X auquel on ajoute une idée Y.

**Formule** : X + Y = … ?

**Exemples** :

- Jeux + Apprentissage d'une langue -> Duolingo
- Psychologie + Économie -> Finance comportementale

## 3. X↑

Trouver tous les problèmes possibles qu'une idée peut résoudre. Une idée qui résout un problème peut en résoudre beaucoup d'autres.

**Formule** : Avec un marteau, trouvez tous les clous

**Exemples** :

- Les serviettes hygiéniques étaient utilisées à l'origine pour panser les blessures des soldats pendant la Seconde Guerre mondiale.
- Le rince-bouche a été conçu à l'origine pour être un désinfectant à utiliser en chirurgie.

## 4. X↓

Trouver toutes les solutions possibles au problème que vous essayez de résoudre.

**Formule** : Avec un clou, trouvez tous les marteaux

## 5. X++

Prendre une idée X et lui ajouter un adjectif (économique, luxueux, écologique, démocratique, social, etc...)

**Formule** : X + Adjectif = … ?

**Exemples** :

- Location de voiture + sociale -> covoiturage
- Voyages-SNCF + simple -> Trainline (Capitaine Train)

## 6. X̄

Faire exactement le contraire de ce que l'on pense.

**Formule** : Faites exactement le contraire

**Exemples** :

- Banque avec plein de bureaux dans le monde -> Banque 100% en ligne
