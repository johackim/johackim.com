Se connecter à un serveur IRC avec Weechat

Mis à jour le dimanche 1 août 2021 par johackim

Installer Weechat

apt install -yq weechat

Démarrez weechat

weechat

Ajoutez un serveur

/server add debian irc.debian.org

Lister les serveurs

/server list

Ajouter un serveur sans SSL

/server add debian irc.debian.org/6667 -notls

Se connecter dessus

/connect debian

Lister les channels

/list

Changer de nom d'utilisateur

/nick <username>

Changer de channel

Pour changer de fenêtre :

  • alt+left
  • alt+right

Références :

  • Linux
