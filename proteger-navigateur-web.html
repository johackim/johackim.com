Protégez votre navigateur web

Mis à jour le mercredi 22 juin 2022 par johackim

Si vous voulez naviguer sur le web sans avoir peur d'être traqué, voici les outils à utiliser.

Firefox

Firefox est un logiciel libre. Contrairement à Chrome, il ne contient pas de code propriétaire.

C'est le navigateur à privilégier si vous désirez naviguer sur le web librement.

Il existe une version pour Linux, Windows, macOS, Android, iOS, etc...

PS : Il existe aussi la version communautaire de Firefox -> LibreWolf

uBlock Origin

Pour éviter les publicités intempestives, le meilleur add-on pour ça, c'est uBlock Origin.

Contrairement à Adblock, uBlock Origin ne se fait pas rémunérer par les géants du Web pour laisser des publicités affichées et il consomme moins de ressources (CPU et RAM).

Ungoogled-chromium

Si comme moi vous trouvez que Firefox est plus lent que Chrome/Chromium, vous pouvez installer la version degooglisé de Chrome.

Vous pouvez l'installer depuis cette page.

Si vous êtes sur Linux, vous pouvez aussi utiliser Flatpak :

flat install com.github.Eloston.UngoogledChromium

PS : Le chrome store n'est pas disponible par défaut, il vous faudra installer cette extension manuellement pour l'utiliser.

Libredirect

Si vous aimez le contenu de YouTube, Instagram, Twitter et compagnie mais que vous ne voulez pas être traqué lors de son visionnage vous pouvez utiliser l'extension Libredirect.

Une fois installé, tous les contenus seront automatiquement démarrés sur des alternatives respectueuses de la vie privée :

  • YouTube => Invidious
  • Twitter => Nitter
  • Instagram => Bibliogram
  • Reddit => Libreddit
  • Imgur => Rimgo
  • Google => SearX
  • Translate => SimplyTranslate
  • Maps => OpenStreetMap
