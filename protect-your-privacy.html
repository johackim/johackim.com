Free Softwares

  • Use alternatives to proprietary softwares - alternativeto.net

NOTE: Free Software gives everybody the right to use, study, share and improve software. This right helps support other fundamental freedoms like freedom of speech, press and privacy.

Operating System

  • Disable your webcam - disable-webcam.sh
  • Don't use Windows 10, It's a privacy nightmare
  • Keep your OS and apps update
  • Use open-source operating system
  • Use a secure operating system - Tails OS or Qubes OS

Phone

  • Use secure SMS application - Signal
  • Use open-source Operating system - LineageOS
  • Detect attacks of fake base stations (IMSI-Catcher) on your GSM/UMTS network - AIMSICD
  • Android application to monitor your physical room in your absence - haven

Passwords

  • Check if you have an account that has been compromised in a data breach - haveibeenpwned.com
  • Use a stateless password management solution - Master Password

Block advertising

  • Set unified hosts file to block fakenews, gambling, porn, social, adware & malware - hosts
  • Install a browser plugin to prevent third parties from tracking you - uBlock Origin

Browser

  • Use open-source browser - Firefox or Tor
  • Configure firefox to enhance the privacy of your browser
  • Test whether your browser setup is safe against tracking - Panopticlick

Mail

  • Secure your communications - Protonmail
  • Use disposable email addresses

VPN

  • Use a vpn solution
  • Disable IPV6 if your VPN no support it - Check IPV6 leak
  • Disable WebRTC in Firefox - Check WebRTC leak
  • Use a different DNS server than your ISP - Check DNS leak
  • Disable browser location

Anti-Forensic

  • Wipe unused old computers, phones, tablets, hard drives, etc - shred -n 5 -vz /dev/sda
  • Use a cleaner - Bleachbit

Encryption

  • Encrypt your hard drive - LUKS

Backups

  • Frequently backup your computer - borg
