## johackim.com

My personal website

## ✨ Features

- [x] Wikilinks
- [x] Mermaid
- [x] Sitemap
- [x] robots.txt
- [x] Github Pages
- [x] Custom 404
- [x] Tailwind CSS
- [x] Meta tags
- [x] RSS Feed
- [x] No javascript
- [x] Favicon
- [x] Newsletter
- [x] Responsive
- [x] Comments
- [x] Callouts
- [x] Analytics
- [ ] Search
- [ ] Dark Mode
- [ ] Graph Viewer
- [ ] Member access
- [ ] Security headers
- [ ] Table of content

## 💻 Usage

```bash
npm install
npm run dev
```
