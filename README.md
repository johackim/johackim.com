# johackim.com

My personal digital garden.

## Features

- [x] Search
- [x] Obsidian Markdown syntax
- [x] MDX support
- [x] Dark Mode
- [x] Newsletter
- [x] Custom SEO
- [x] 100% Lighthouse performance score
- [x] Tailwind CSS
- [x] Posthog support
- [x] RSS Feed
- [x] Mermaid support
- [ ] Graph Viewer

## Quick start

```bash
# Install dependencies
yarn

# Run the app
npm run dev

# Go to http://localhost:3000
```

## Running the tests

```bash
npm test
```
