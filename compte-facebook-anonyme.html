Créer un compte facebook anonyme

Mis à jour le mardi 11 mai 2021 par johackim

Détenir un compte Facebook anonyme est beaucoup plus complexe qu'avoir un compte Google anonyme.

Pour créer un compte Facebook anonyme, il faut passer par 6 étapes :

  1. Générer une fausse photo de profil
  2. Générer un faux nom
  3. Créer un compte e-mail protonmail
  4. Utiliser une adresse IP résidentielle
  5. Accéder à la version mobile de Facebook
  6. S'inscrire avec votre fausse identité

1. Générer une fausse photo de profil

Téléchargez une photo sur ce générateur de photo de profil, cropez l'image et changez sa couleur, son orientation et sa taille.

2. Générez un faux nom

Utilisez un générateur de faux nom comme Fakenamegenerator pour trouver une idée de faux nom à définir sur votre profil Facebook.

3. Créez-vous un compte protonmail

Créez une adresse email sur Protonmail.

4. Utilisez une adresse IP résidentielle

Ne passez pas par un VPN, utilisez de préférence votre adresse IP.

Vous pouvez vérifier la qualité de votre IP sur un service comme scamalytics avant de vous inscrire.

5. Accédez à la version mobile de Facebook

Rendez-vous sur la version mobile de Facebook.

De préférence avec le navigateur d'un smartphone (ex: Opera).

6. Inscrivez-vous avec votre fausse identité

Inscrivez-vous avec les informations précédemment générées :

  • Photo
  • E-mail
  • Faux nom

Si vous ne passez pas minutieusement par toutes ses étapes, Facebook va stopper votre compte et vous demander une carte d'identité.

C'est le seul moyen que j'ai trouvé pour disposer d'un compte Facebook.

PS: Après inscription, pensez à effectuer des actions qu'un utilisateur lambda ferait ; liker, commenter, s'abonner à des pages, ect...
